# Support

## Getting Help

### Documentation
- [README](../README.md) — Project overview and quick start
- [docs/](../docs/) — Detailed documentation

### Community
- **Issues** — Use [GitHub Issues](https://github.com/mohamedsillahhh-cloud/DevFlow/issues) for bug reports and feature requests
- **Discussions** — Start a [GitHub Discussion](https://github.com/mohamedsillahhh-cloud/DevFlow/discussions) for questions and ideas

### Before Opening an Issue
1. Check the [FAQ](../docs/faq.md) for common questions
2. Search existing [issues](https://github.com/mohamedsillahhh-cloud/DevFlow/issues) for similar problems
3. Make sure you have the latest version

### Bug Reports
Include the following in your bug report:
- DevFlow version (commit hash or release)
- Browser and OS
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)

### Feature Requests
Describe what you'd like to see, why it's useful, and any implementation ideas you have.
